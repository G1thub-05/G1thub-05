Here is the complete list of all WinGet commands you can use in the Windows Command Prompt (CMD), categorized by what they do.
## Managing Applications (Install, Update, Remove)

* winget install <app>: Installs a specific application.
* winget upgrade <app>: Updates a specific application to the latest version.
* winget upgrade --all: Updates all installed applications at once.
* winget uninstall <app>: Completely removes an application from your PC.

## Finding and Viewing Applications

* winget search <name>: Searches for an app in the repository to find its exact ID.
* winget show <app>: Displays detailed information about a specific application.
* winget list: Shows a list of all applications currently installed on your PC.
* winget features: Lists experimental features available in your version of WinGet.

## Backup and Restore (Import/Export)

* winget export -o <filepath.json>: Backs up your list of installed apps to a file.
* winget import -i <filepath.json>: Installs all apps listed in your backup file automatically.

## Advanced and System Settings

* winget settings: Opens the settings configuration file (JSON format).
* winget source list: Shows the repositories where WinGet looks for applications.
* winget pin list: Displays a list of apps that you have blocked from updating.
* winget validate <manifest>: Checks if an application manifest file is valid (for developers).

## Useful Combination Commands

+-------------------------------------------+--------------------------------------------------------------+
| Command                                   | What it does                                                 |
|-------------------------------------------|--------------------------------------------------------------|
| winget upgrade --all --uninstall-previous | Updates everything and deletes old versions.                 |
| winget upgrade --all --silent             | Updates all apps silently in the background.                 |
| winget install <app> --interactive        | Shows the normal setup wizard window during install.         |
| winget pin add <app>                      | Locks an app to its current version so it won't auto-update. |
+-------------------------------------------+--------------------------------------------------------------+

Would you like to know how to create a one-click desktop shortcut to update all your apps, or do you need help fixing an error with a specific WinGet command...? 



Here is how to create a one-click desktop shortcut to update all your apps automatically using WinGet.

## Step 1: Create the Shortcut

   1. Right-click on any empty space on your desktop.
   2. Hover over New and select Shortcut.

## Step 2: Paste the Command

   1. In the box that says "Type the location of the item", paste this exact code:
   
   cmd.exe /c winget upgrade --all --uninstall-previous --silent
   
   2. Click Next.

## Step 3: Name the Shortcut

   1. Type a name like Update All Apps.
   2. Click Finish.

## Step 4: Make it Run as Administrator (Important)

   1. Right-click your new desktop shortcut and select Properties.
   2. Click the Advanced... button (under the Shortcut tab).
   3. Check the box for Run as administrator.
   4. Click OK, then click Apply and OK.

Now, whenever you double-click this shortcut, it will silently update all your apps and remove older versions in the background.

Here is how to change the shortcut icon and schedule the update process to run automatically every week.
## Part 1: Change the Shortcut Icon

   1. Right-click your "Update All Apps" desktop shortcut and select Properties.
   2. Go to the Shortcut tab and click the Change Icon... button.
   3. Erase whatever text is in the top box and paste this exact file path to open Windows system icons:
   
   shell32.dll
   
   4. Press Enter.
   5. Scroll to the right, select the green circular arrows icon (or any update icon you prefer), and click OK.
   6. Click Apply and then OK.

------------------------------
## Part 2: Schedule it to Run Automatically Every Week
You can use the built-in Windows Task Scheduler so you never have to click the shortcut manually again.

   1. Press the Windows Key, type Task Scheduler, and press Enter.
   2. On the right side, click Create Basic Task....
   3. Name it Weekly App Update and click Next.
   4. Select Weekly, click Next, and choose your preferred day and time (e.g., Sunday at 12:00 PM). Click Next.
   5. Select Start a program and click Next.
   6. Fill out the boxes exactly like this:
   * Program/script: cmd.exe
      * Add arguments: /c winget upgrade --all --uninstall-previous --silent
   7. Click Next, check the box that says Open the Properties dialog for this task when I click Finish, and click Finish.
   8. In the new properties window that pops up, check the box at the bottom for Run with highest privileges (this ensures it     
      runs as Administrator).
   9. Click OK.
